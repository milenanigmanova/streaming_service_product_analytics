# Анализ и декомпозиция метрик продукта (стриминговый сервис)

## Описание
Проект по анализу рынка и построению системы продуктовых метрик для стримингового сервиса (аналог Netflix / Кинопоиск).

## Цель
Сформировать систему метрик и определить точки роста продукта на основе анализа рынка и аудитории.

## Задачи
- Провести анализ рынка и конкурентов  
- Описать целевую аудиторию  
- Построить CJM  
- Сформулировать позиционирование и УТП  
- Разработать тарифную модель  
- Определить ключевую метрику продукта  
- Построить систему метрик  
- Выявить риски и точки роста  

## Стек
Product Analytics, Excel / Google Sheets, CJM, продуктовые метрики (Retention, LTV, DAU/MAU)

## Результат
- Выбрана ключевая метрика — Retention  
- Построена система метрик, связанная с бизнес-целями  
- Выявлены риски и ограничения  
- Сформулированы рекомендации по росту продукта (контент, ценообразование, удержание пользователей)
